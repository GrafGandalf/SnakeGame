Порядок установки для разработки:

Для разработки были использованы:
Xcode Version 13.2.1 (13C100)
CMake Version 3.23.1 

1. Скачайте архив на свой компьютер.
2. Разархивируйте в любую удобную вам папку.
3. В приложении CMake выберите папку с проектом и папку в которую вы хотите сгенерировать проект.
4. Сгененрируйте под свой компилятор, после чего откройте проектный файл (Для Xcode Game.xcodeproj)
5. В строках с 11 по 28 в Main.cpp замените ссылки текстур и шрифтов. (Правая кнопка -> alt+Скопировать путь).
6. Наслаждайтесь.
